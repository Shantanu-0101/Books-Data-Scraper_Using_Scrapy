# Books-Data-Scraper_Using_Scrapy
A high-performance web scraper extracting book details from e-commerce sites

## Key Features: 
Automatically crawls through paginated book listings 
Extracts titles, prices, and star ratings with 99% accuracy 
Cleans and transforms raw data into analysis-ready formats 

## Tech Stack: 
Python • Scrapy • Pandas • XPath/CSS Selectors • CSV/JSON Export
